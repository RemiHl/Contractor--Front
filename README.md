## Contractor - Frontend

Ce dépôt contient le frontend de Contractor, une plateforme de génération de contrats destinée aux freelances, entrepreneurs, et petites entreprises. Le frontend est construit avec React et Vite pour une expérience utilisateur rapide et moderne.

# Aperçu du projet
Contractor simplifie la création, la personnalisation et la gestion des contrats. Ce frontend offre une interface intuitive pour s’inscrire, se connecter, et utiliser les différentes fonctionnalités de génération et de personnalisation de documents juridiques.

# Fonctionnalités du Frontend
-Page d'accueil : Introduction aux fonctionnalités principales de la plateforme.
-Formulaire d'inscription : Création d'un compte utilisateur avec validation de l'email et du mot de passe.
-Formulaire de connexion : Authentification sécurisée avec gestion des erreurs.
-Tableau de bord utilisateur (à venir) : Interface pour accéder aux modèles de contrats, personnaliser les documents, gérer les contrats enregistrés.
-Effets visuels : Animation d’ombre qui suit la souris et sections interactives pour améliorer l'expérience utilisateur.

# Technologies utilisées
-React avec Vite pour un rendu rapide et une architecture modulaire.
-CSS pour le stylisme, avec un design moderne et réactif.
-React Router pour la gestion des routes (navigation entre les pages).
-Hooks React pour la gestion de l'état et les effets interactifs (comme l'effet de suivi de la souris).
